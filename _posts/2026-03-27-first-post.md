# 첫 번째 포스트

블로그를 시작합니다. Jekyll과 GitHub Pages를 이용해 기술 블로그를 운영할 예정입니다.

## Jekyll이란?

Jekyll은 정적 사이트 생성기로, Markdown 파일을 HTML로 변환해줍니다. GitHub Pages와 완벽하게 통합되어 별도의 서버 없이 블로그를 운영할 수 있습니다.

## 앞으로의 계획

- 개발 경험 공유
- 기술 스택 소개
- 문제 해결 과정 기록
