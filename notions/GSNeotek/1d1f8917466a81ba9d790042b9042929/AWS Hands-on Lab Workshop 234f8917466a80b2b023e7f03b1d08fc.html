Amazon Q in Connect 워크샵

Amazon Q in Connect를 활성화하면 하나의 지식베이스와 연결된 어시스턴트인 Q in Connect 도메인을 생성합니다. Q in Connect 도메인 사용에 관한 정보는 다음과 같습니다.

  • 각각의 Amazon Connect 인스턴스는 하나의 도메인에만 연결될 수 있습니다.
  • 하나의 도메인은 AI Agent 구성을 통해 여러 개의 지식 베이스를 지원할 수 있습니다.
  • 여러 개의 도메인을 생성할 수 있지만, 도메인 간에는 외부 애플리케이션 통합이나 고객 데이터를 공유하지 않습니다.
  • 생성하는 모든 외부 애플리케이션 통합은 도메인 수준에서 이루어집니다. 도메인과 연결된 모든 Amazon Connect 인스턴스는 해당 도메인의 통합을 상속받습니다.
  • 다른 도메인을 선택함으로써 언제든지 현재 도메인에서 새로운 도메인으로 Amazon Connect 인스턴스의 연결을 변경할 수 있습니다.
