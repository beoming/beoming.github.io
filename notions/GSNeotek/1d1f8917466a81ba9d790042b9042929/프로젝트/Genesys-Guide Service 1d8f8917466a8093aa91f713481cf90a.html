🧩 구성 예시

1. 📘 시나리오 기반 플로우 가이드

예제 시나리오:

📞 “전화가 들어오면 특정 큐로 연결하고 녹취까지 저장하려면?”

✅ 화면 구성

  • 좌측: 시나리오 목록 (예: 인바운드 설정, 녹취 설정, IVR 흐름 구성 등)
  • 우측:
    1. 간단한 Flow Chart (react-flow, mermaid.js 등)
    1. 단계별 가이드 (번호 + 설명 + 스크린샷)
    1. 필요한 경우 API 예시 (단순 REST 문서 형식)

[Architect 흐름 설정]

   ↓

2. 📷 GUI Matching Help

예제: "Edge 장비를 등록하고 SIP 트렁크를 설정하고 싶어요"

✅ 화면 구성

  • 실제 Genesys UI 캡처 이미지
  • 이미지 위에 툴팁/하이라이트: “여기를 클릭하세요”, “이 메뉴를 찾아보세요”
  • 예시 툴:
    • react-tooltip
    • react-image-mapper
    • 간단한 커스텀 SVG 박스 + 설명

✨목표 설정

  • 단순한 UI 구성으로 신규 유저 및 교육생도 빠르게 접근 가능
  • 문서 없이도 따라만 하면 되는 단계별 설명
  • Heavy한 기술 없이도 실무자에게 충분한 가치 제공
  • 초기에는 Markdown 또는 JSON 기반으로 가볍게 구성 → 추후 CMS 연동도 가능
