Agent Assist integration with Salesforce

아키텍처 관점에서 구성 요소

구조는 크게 3개 영역으로 나뉩니다.

1) Google Cloud

여기에는 백엔드 인프라가 있습니다.

예:

  • UI Connector
  • Integration Backend
  • 인증/이벤트 처리
  • Agent Assist 기능과 연결되는 서버 구성

즉 Agent Assist의 실제 AI 기능과 백엔드 연동은 Google Cloud 쪽에서 담당합니다.

2) Salesforce Instance

상담사가 실제로 일하는 환경입니다.

여기에는:

  • Agent Assist UI Modules
  • 이를 감싸는 LWC
  • Salesforce Agent Desktop

이 있습니다.

즉 Salesforce 안에서는 LWC가 Agent Assist UI를 호스팅하는 접점 역할을 합니다.

3) Web Client

최종 고객이 사용하는 채팅 클라이언트입니다.

고객은 웹 채팅으로 문의하고,

그 메시지가 Salesforce와 연결되며, 상담사는 Salesforce Agent Desktop에서 응대합니다.

4. Salesforce와 Agent Assist가 연결되는 방식

핵심 포인트는 이것입니다.

  • Agent Assist UI Modules는 공통 모듈
  • Salesforce에서는 이 모듈을 바로 쓰지 않고
  • Lightning Web Component가 wrapper 역할을 함
  • 이 wrapper가 Salesforce 환경에 맞게 UI를 붙여줌

그리고 메시징은:

  • Messaging for In-App and Web
  • Omni-Channel Messaging

과 연결됩니다.

즉 Salesforce 채팅 플랫폼과 Agent Assist UI 사이를

LWC + UI Connector가 이어주는 구조입니다.

5. 실제 동작 예시

영상의 예시는 상담 흐름 안에서 Agent Assist가 어떻게 보이는지를 보여줍니다.

예를 들면:

고객이 “hi”라고 말함

  • 상담사는 Smart Reply 추천을 받음

고객이 co-insurance deductible 문의

  • 추가 Smart Reply 추천 제공

고객이 membership card 재발급 요청

  • GKA(Generative Knowledge Assist) 제안 표시

상담사가 요약 버튼 클릭

  • Summarization 생성

또는 채팅 종료 시 자동으로 요약 생성

요약에는 보통 이런 필드가 포함됩니다.

  • Situation
  • Action
  • Resolution
  • Customer Satisfaction

이 필드들은:

  • 수정 가능
  • 저장 가능
  • 복사 가능

또한 AI Coach가

  • 주소 최신 여부 확인
  • 전화번호 최신 여부 확인

같은 추가 코칭 제안을 줄 수 있습니다.

즉 단순 답변 추천만이 아니라,

답변 추천 + 지식 제안 + 요약 + 코칭

까지 Salesforce 화면 안에서 제공됩니다.

6. LWC란 무엇인가

LWC는 Salesforce의 현대적인 프론트엔드 컴포넌트 프레임워크입니다.

설명대로:

  • HTML 템플릿
  • CSS 모듈
  • JavaScript 클래스 기반 컴포넌트

구조를 사용합니다.

React, Angular, Vue 경험이 있으면 익숙하게 느껴질 수 있습니다.

즉 Salesforce 전용이긴 하지만, 개념적으로는

현대 JS 컴포넌트 프레임워크와 유사한 방식입니다.

Salesforce 쪽에서 Agent Assist가 붙는 방식

이 모듈에서는 정의된 custom LWC component가 Agent Assist UI Modules를 wrapper 형태로 감싼다고 설명합니다.

즉:

  • Agent Assist 기능 자체를 Salesforce 코드로 새로 구현하는 것이 아니라
  • 기존 UI Modules를 LWC wrapper로 포장해서
  • Salesforce Agent Desktop에 꽂는 방식입니다.

여기에 붙일 수 있는 대표 기능:

  • Summarization
  • Smart Reply
  • 기타 Agent Assist 기능들
