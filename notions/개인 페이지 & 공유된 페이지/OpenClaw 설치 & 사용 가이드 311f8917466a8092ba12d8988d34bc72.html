OpenClaw 설치 & 사용 가이드

1. 오픈클로(OpenClaw)가 대체 뭔가요?

1-1. 한 줄 요약

OpenClaw는 내 컴퓨터에 설치하는 개인 AI 비서입니다. 텔레그램, 왓츠앱, 디스코드 같은 메신저로 AI에게 말을 걸면, AI가 내 컴퓨터에서 직접 파일 정리, 웹 검색, 일정 관리, 코드 실행 등을 대신 해줍니다.

1-2. 쉬운 비유로 이해하기

영화 아이언맨의 자비스(JARVIS)를 떠올려보세요. 토니 스타크가 “자비스, 이것 좀 해줘”라고 말하면 자비스가 알아서 처리하죠? OpenClaw가 바로 그 자비스입니다.

비교 항목설명
ChatGPT / Claude 웹대화만 가능 ("말만 하는 AI")
시리 / 빅스비알람 설정 정도만 가능 (권한 제한)
OpenClaw내 컴퓨터를 실제로 조작하는 AI ("일하는 AI")

1-3. 왜 지금 OpenClaw가 중요한가?

  • AI 에이전트(Agent) 시대의 개막: 2024~2025년은 AI가 ‘대답만 하는 단계’에서 ‘실제로 행동하는 단계’로 넘어가는 전환점입니다. OpenClaw는 이 전환을 가장 먼저, 가장 쉽게 체험할 수 있는 오픈소스 도구입니다. GitHub 스타 10만 개를 돌파하며 역대 가장 빠른 성장세를 기록 중입니다.
  • 왜 비전공자에게도 중요한가?

    이제 AI는 단순히 질문에 답하는 것을 넘어, 여러분 대신 이메일을 보내고, 파일을 정리하고, 뉴스를 요약하고, SNS에 글을 올리는 시대가 열리고 있습니다. OpenClaw를 지금 경험해두면, 앞으로 모든 사람이 사용하게 될 ‘AI 비서 시대’를 미리 준비하는 셈입니다.

핵심 포인트

OpenClaw = 내 컴퓨터에 사는 AI 비서. 메신저로 말 걸면 진짜로 일해줌. 오픈소스라 완전 무료. 다만 AI 두뇌(API) 연결 비용은 별도 (월 몇천원 수준).

1-5. OpenClaw의 이름 변천사

OpenClaw는 원래 Clawdbot(클로드봇) 이라는 이름으로 시작했습니다. Anthropic(Claude 개발사)의 요청으로 Moltbot(몰트봇) 으로 변경되었다가, 현재의 OpenClaw(오픈클로) 라는 이름으로 최종 정착했습니다. 인터넷에서 이전 이름으로 검색해도 같은 프로젝트입니다.

B. 텔레그램 계정 (메신저 연동용)

OpenClaw와 대화하려면 메신저가 필요합니다. 초보자에게는 텔레그램(Telegram)이 가장 쉽습니다. 스마트폰에 텔레그램 앱을 설치하고 계정을 만들어두세요. PC에도 텔레그램 데스크톱 앱을 설치해두면 편합니다.

보안 경고 (매우 중요!)

OpenClaw는 실험적 소프트웨어입니다. 개발자 본인도 ‘기술에 익숙하지 않은 사람은 설치하지 말라’고 경고했습니다. 반드시 다음을 지켜야 합니다:

  1. 개인 중요 파일이 없는 PC나 가상머신에 설치할 것
  1. API 키를 절대 타인에게 공유하지 말 것
  1. 검증되지 않은 스킬(확장 기능)을 설치하지 말 것

API 키가 유출되면 타인이 사용해 과금될 수 있습니다.

3. WSL2 설치하기 (리눅스 환경 준비)

3-1. WSL이 뭔가요?

WSL(Windows Subsystem for Linux)은 윈도우 안에서 리눅스를 돌릴 수 있게 해주는 마이크로소프트의 공식 기능입니다. OpenClaw는 원래 맥/리눅스용으로 만들어졌기 때문에 윈도우에서는 WSL2를 통해 설치해야 합니다.

쉬운 비유: 윈도우라는 큰 집 안에 ‘리눅스 방’을 하나 만드는 겁니다. OpenClaw는 이 리눅스 방에서 생활하게 됩니다.

3-2. 설치 과정 (5분 소요)

STEP 1) PowerShell을 관리자 모드로 실행

  1. Windows 검색창(돋보기 아이콘)에 PowerShell 입력
  1. ‘Windows PowerShell’ → 마우스 오른쪽 클릭 → 관리자 권한으로 실행

왜 관리자 모드인가요?

WSL 설치는 윈도우 시스템 설정을 변경하는 작업이라 관리자 권한이 필요합니다. 일반 모드로 실행하면 설치가 되지 않습니다.

STEP 2) WSL2 설치 명령어 입력

파란색 PowerShell 창에서 아래 명령어를 복사/붙여넣고 Enter:

wsl --install

WSL2와 Ubuntu 24.04가 자동으로 설치됩니다. 설치에 몇 분이 걸릴 수 있습니다.

STEP 3) 컴퓨터 재시작

설치가 완료되면 컴퓨터를 반드시 재시작해야 합니다. 재시작하지 않으면 WSL이 제대로 작동하지 않습니다.

STEP 4) Ubuntu 초기 설정

재시작 후 Ubuntu 창이 자동으로 열립니다.

(안 열리면 시작 메뉴에서 Ubuntu를 검색해서 실행하세요.)

입력 항목은 아래와 같습니다:

입력 항목설명
username영문 소문자로 원하는 이름 입력 (예: myname)
password비밀번호 설정 (입력 시 화면에 안 보이는 게 정상!)

비밀번호는 안 보입니다. (윈도우 보안 정책 때문임)

리눅스에서 비밀번호 입력 시 보안을 위해 화면에 아무것도 표시되지 않습니다(별표도 안 나옴). 그냥 비밀번호 입력 후 Enter.

STEP 5) 설치 확인

프롬프트(명령어 입력 대기 상태)가 사용자이름@DESKTOP-XXXXX:~$ 형태로 바뀌면 성공입니다.

4. Node.js 설치하기 (OpenClaw의 엔진)

4-1. Node.js가 뭔가요?

쉬운 비유: 자동차에 엔진이 필요하듯, OpenClaw도 돌아가려면 Node.js라는 ‘엔진’이 필요합니다. Node.js는 JavaScript라는 프로그래밍 언어를 실행시켜주는 프로그램입니다. OpenClaw가 JavaScript로 만들어져 있기 때문에 반드시 설치해야 합니다.

4-2. 설치 과정

Ubuntu 창(까만 터미널 창)에서 아래 명령어를 한 줄씩 실행합니다.

STEP 1) 시스템 업데이트

sudo apt update && sudo apt upgrade -y

sudo는 ‘관리자 권한으로 실행해줘’라는 뜻입니다. 비밀번호를 입력하라고 나올 수 있습니다.

STEP 2) Node.js 22 설치

curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt install -y nodejs

STEP 3) 설치 확인

node --version

v22.x.x 같은 버전 번호가 나오면 성공입니다.

Ubuntu(WSL) 창에 붙여넣기 방법

Ubuntu(WSL) 터미널에서는 Ctrl+V가 아니라 마우스 오른쪽 클릭으로 붙여넣기 합니다! 또는 Ctrl+Shift+V 사용 가능.

5. OpenClaw 설치하기

5-1. 설치 명령어

Ubuntu 창에서 아래 명령어를 입력합니다.

npm install -g openclaw@latest
  • npm은 Node.js의 패키지(프로그램) 설치 도구입니다.
  • g는 ‘전체(global)에 설치해줘’라는 뜻입니다.
  • openclaw@latest는 ‘오픈클로 최신 버전’이라는 뜻입니다.

STEP 확인) 설치 확인

openclaw --version

openclaw/2026.x.x 같은 버전 정보가 나오면 설치 성공입니다

에러가 발생했다면?

  • EACCES 에러: sudo npm install -g openclaw@latest 로 재시도
  • network 에러: 인터넷 연결 확인
  • node: command not found: Node.js 설치부터 다시 진행

7. 텔레그램 봇 만들기

7-1. 왜 텔레그램인가요?

OpenClaw와 대화하려면 메신저 채널이 필요합니다. 텔레그램은 봇 생성이 가장 간단하고, 초보자에게 가장 추천됩니다. 카카오톡처럼 개인 메신저 앱인데, 봇(자동 응답 프로그램)을 무료로 만들 수 있다는 점이 특별합니다.

7-2. 봇 만들기 과정

STEP 1) 텔레그램에서 @BotFather 찾기

  1. 텔레그램 앱(스마트폰 또는 PC) 실행
  1. 검색창에 @BotFather 입력
  1. 파란색 체크 표시가 있는 공식 계정 선택
  1. 대화 시작

STEP 2) /newbot 명령어 입력

채팅창에 /newbot 입력 후 전송

BotFather가 봇의 이름을 물어봅니다.

STEP 3) 봇 이름 설정

  1. 먼저 ‘표시 이름’ 입력 (예: My AI Assistant)
  1. 그 다음 ‘사용자명’ 입력 → 반드시 bot 으로 끝나야 함 (예: myai_helper_bot)

STEP 4) 봇 토큰 복사

BotFather가 123456789:ABCdefGHIjkl... 형태의 긴 문자열을 보내줍니다.

이것이 봇 토큰(Bot Token)입니다. 복사해서 메모장에 안전하게 저장하세요. 온보딩에서 필요합니다.

봇 토큰도 비밀번호와 같습니다

유출되면 남이 내 봇을 제어할 수 있습니다. API 키와 마찬가지로 절대 공개하면 안됩니다.

8. OpenClaw 초기 설정 (온보딩)

8-1. 온보딩 마법사 시작

이제 모든 재료가 준비되었고, Ubuntu 창에서 아래 명령어를 입력하세요.

openclaw onboard --install-daemon
  • -install-daemon은 “컴퓨터가 켜질 때마다 OpenClaw가 자동으로 시작”되게 해주는 옵션입니다. 이걸 붙여야 매번 수동으로 실행할 필요가 없습니다.

온보딩 마법사가 시작되면 아스키 아트와 함께 환영 메시지가 나타납니다. 여러 가지 질문이 순서대로 나옵니다.

온보딩 시 생성된 게이트웨이 토큰을 입력하면 대시보드에서 바로 AI와 대화할 수 있습니다.

토큰 확인:

cat ~/.openclaw/openclaw.json

예시:

"gateway": {
  "token": "여기에_긴_문자열"
}

주소창에 이렇게 입력:

  • http://127.0.0.1:18789/?token=YOUR_TOKEN

이제 일반적인 웹사이트처럼 소통이 가능합니다.

추가: 외부 API 확장 아이디어

봇의 이름을 설정해주면 시작입니다.

Claude는 기본 모델 성능만으로도 강력하지만, 외부 API를 추가로 연동하면 기능을 훨씬 확장할 수 있습니다.

검색, 시장 데이터, 온체인 분석, 자동화, 음성 생성 등 필요한 기능을 선택적으로 붙여 전문 리서치 도구나 자동화 봇 수준으로 강화할 수 있습니다.

다만 API는 대부분 별도 과금 구조이므로, 실제로 자주 사용할 기능만 추가하는 것이 효율적입니다.

아래는 사람들이 실제로 많이 붙여서 사용하는 대표 API들만 정리한 것입니다.
